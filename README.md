# 🎧 Responsive Headphones Landing Page

This project is a **fully responsive landing page** for a fictional headphone brand. The webpage is designed to switch to **desktop**, **tablet**, and **mobile** screen sizes — using **pure HTML and CSS only**, with a strong focus on accessibility and responsive web design best practices.

---

## 📱 Screens Supported

| Device Type | Max Width | Version Triggered |
|-------------|-----------|--------------------|
| Desktop     | > 1024px  | Default View       |
| Tablet      | ≤ 768px   | Responsive Layout  |
| Mobile      | ≤ 480px   | Mobile-First View  |

---

## 💡 Features

- ✅ Pure HTML & CSS (No JavaScript)
- ✅ Responsive layout using media queries
- ✅ Hover and active states for links & buttons
- ✅ Fully accessible semantic HTML structure
- ✅ Max content width: **1000px**, centered

---

## 🎨 Design Specifications

| Element         | Behavior                                  |
|----------------|--------------------------------------------|
| `a:hover/active`   | Color: `#FF6565`                         |
| `button:hover/active` | `opacity: 0.9`                        |
| Max Width       | `1000px` (centered with margin auto)      |
| Mobile Switch   | Triggered at screen width ≤ `480px`       |
